macro_rules! trace {
    ($($arg:tt)*) => (
        if !cfg!(any(log_level = "off",
                     log_level = "error",
                     log_level = "warn",
                     log_level = "info",
                     log_level = "debug")) {
            log!($crate::LogLevel::Debug, $($arg)*);
        }
    )
}

Logs a message at the trace level.

Logging through this macro is disabled if any of the following cfgs are present: log_level = "off", log_level = "error", log_level = "warn", log_level = "info", or log_level = "debug".
