pub enum MaybeSslStream<S> where S: Stream {
    Ssl(SslStream<S>),
    Normal(S),
}

A utility type to help in cases where the use of SSL is decided at runtime.

Variants

Ssl

A connection using SSL

Normal

A connection not using SSL
