A set of errors that can occur parsing HTTP streams.

Variants

An invalid Method, such as GE,T.

HttpUriError

HttpVersionError

HttpHeaderError

HttpStatusError

An invalid Status, such as 1337 ELITE.

HttpIoError

An IoError that occured while trying to read or write to a network stream.
